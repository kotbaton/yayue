Separate subfolders should be created here for specific collections of of tools and
the corresponding *.py sources and data.
The details about the sub-folders contents should be provided in the Readme.txt
file in each sub-dir.

The content of the sub-dirs:
- techLearn20: clustering of the technology-learning alternatives for diverse
  learning rates,
- wti_prices: analysis of the WTI crude-oil prices
